# RAG Document Assistant — Frontend

A minimal React chat UI that talks to your n8n RAG workflow. Built with Vite.

See the full step-by-step guide in the chat conversation where this project was generated.
Quick reference:

1. `npm install`
2. `cp .env.example .env` and paste your n8n webhook URL into it
3. `npm run dev` — open http://localhost:5173
4. `npm run build` before deploying
5. Push to GitHub, import into Vercel, set `VITE_N8N_WEBHOOK_URL` as an Environment Variable in the Vercel project settings
